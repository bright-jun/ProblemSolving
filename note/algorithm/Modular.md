# Modular

## 큰 수의 나머지 구하는 알고리즘

```
(A + B) % C = ((A % C) + (B % C)) % C
(A - B) % C = ((A % C) - (B % C)) % C
(A * B) % C = ((A % C) * (B % C)) % C
```
